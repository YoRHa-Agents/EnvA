Shared shell preview for landing, demo, and embedded web.

This page records the visual migration from Enva's previous branded Pages shell to the neutral documentation shell used by `RustWebAppCommon`.

Adopted design language

  • Neutral dark tokens: `bg-canvas`, `bg-panel`, `line-soft`, `fg-primary`.
  • System sans body copy, Georgia headings, mono code surfaces.
  • `shell`, `masthead`, `topnav`, `status-strip`, and bordered panel composition.
  • Shared button, card, and command presentation across landing and docs surfaces.

Guardrails that stay intact

  • `exportSelectionList`, `exportSelectionSummary`, and `importConflictSummary` remain stable.
  • Embedded web still owns `remote-option-list`, `remote-option-copy`, and `remoteActionError`.
  • Port helpers in embedded `index.html` keep their current JS strings for unit tests.
  • Pages demo and embedded web remain visually closer without flattening product behavior.
